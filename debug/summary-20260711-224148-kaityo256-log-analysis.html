📚 背景 — このリポジトリは何か

著者が「後で自分で読み返すため」に2018年11月から毎日つけている公開日記で、書かなかった日も NOP と明記する徹底ぶり(221件)。Markdown を push すると GitHub Actions が pandoc で HTML に変換し、GitHub Pages に自動公開される。日記そのものに加えて、この「makefile + pandoc + Actions」という最小構成の静的サイトの作り方自体もひとつの知見になっている。

🧩 ④ 輪講の論文・教科書リスト

研究室ミーティングの回では読んだ論文が著者・年つきの箇条書きで残っており、約162件を機械的に抽出できる。分野は統計力学・機械学習・流体・スピン系・ネットワーク科学と幅広く、「物理系研究室がいま何を読んでいるか」のサンプルになる。教科書輪読も章単位で記録されていて(田崎『統計力学』、佐々『熱力学入門』など)、統計力学の学習カリキュラムとしてそのまま使える。

🎯 まとめ — 自分の日記に活かせること

この日記が7年半後に「掘れる資産」になっているのは、次の3つの型を守っているからと言える。(1) エラーは本文を貼り、解決を1〜3行で書く。(2)「査読」「〜字。」のような定型表現で書くので、後から grep で数えられる。(3) 書かない日も NOP と残すので、時系列が途切れない。同じ形式で日記をつけるなら、この3点をまねるのが一番効く。
